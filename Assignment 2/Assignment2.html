  1. HTML (HyperText Markup Language)
    2. Tech Stack Name Primary Use Cases Key Features/Benefits
    HTML - Structure and content of web pages - Defines the structure of web documents
    - Semantically markup content - Easily readable by humans
    - Essential for building web pages - Compatible with various browsers
    - Supports multimedia elements (images, videos, audio) - Enables accessibility features for all users
    - Integrates with CSS and JavaScript for interactivity
  2. CSS (Cascading Style Sheets)
    3. Tech Stack Name Primary Use Cases Key Features/Benefits
    CSS - Styling and layout of web pages - Separates content from presentation
    - Creating responsive web designs - Allows for consistent styling across multiple pages
    - Customizing the look and feel of web elements - Flexibility through selectors, classes, and IDs
    - Animation and transitions - Media queries for responsive design
    - Preprocessors like SASS or LESS for enhanced efficiency
  3. JavaScript
    4. Tech Stack Name Primary Use Cases Key Features/Benefits
    JavaScript - Adding interactivity to web pages - Client-side scripting language
    - Building web applications and games - Event-driven programming
    - Handling user input and form validation - Wide range of libraries and frameworks available
    - Asynchronous programming for AJAX requests - Cross-platform compatibility
    - Modern ES6+ features for enhanced development
