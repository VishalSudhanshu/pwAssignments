Difference between HTML and HTML5

HTML HTML 5
It did not support audio and video without the use of flash player support. It supports audio and video controls with the use of <audio> and <video> tags.
It uses cookies to store temporary data It uses SQL databases and application cache to store offline data.
Older version of HTML are less mobile-friendly. HTML5 language is more mobile-friendly.
It is almost impossible to get true GeoLocation of user with the help of browser. One can track the GeoLocation of a user easily by using JS GeoLocation API.
Semantic tags like nav, header were not present. New semantics tags for web structure like <nav>, <header>, <footer> etc.
